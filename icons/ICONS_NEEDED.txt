Icons Required for Extension
==============================

You need to create three PNG icons:

1. icon16.png  - 16x16 pixels
2. icon48.png  - 48x48 pixels  
3. icon128.png - 128x128 pixels

Simple Ways to Create Icons:
-----------------------------

Option 1: Use an online tool
- Visit: https://favicon.io/favicon-generator/
- Or: https://www.canva.com/
- Create a simple blue/purple chart icon (ðŸ“Š)

Option 2: Use existing emoji/image
- Find a chart/analytics emoji or icon online
- Resize to required dimensions
- Save as PNG

Option 3: Use Preview (Mac) or Paint (Windows)
- Create new images with required dimensions
- Add text "YT" or draw simple chart icon
- Save as PNG

Temporary Workaround:
--------------------
The extension will still work without icons, but Chrome will show 
a default puzzle piece icon instead.

Place the icons in this directory (icons/) when ready.
