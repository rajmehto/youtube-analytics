üìñ How to Get a Free API Key

Step-by-Step Instructions:

  1. Go to Google Cloud Console
  2. Create a new project or select an existing one
  3. Enable "YouTube Data API v3" for your project
  4. Go to "Credentials" and click "Create Credentials"
  5. Select "API Key"
  6. Copy the generated API key
  7. Paste it above and click "Save API Key"

Note: The free tier includes 10,000 quota units per day, which is enough for regular usage.

Local Data Storage

The extension stores analytics data locally in your browser using IndexedDB. This includes:

  • Video performance history
  • Channel analytics snapshots
  • Keyword ranking trends
  • User preferences

Privacy: All data is stored locally. Nothing is sent to external servers.
